# 시장환경조사 데이터 카테고리 정의서

> 버전 v0.1 · 작성일 2026-06-14 · 용도: 에이전트 조사 항목 정의 + 유사도 산출 기준

---

## 0. 읽는 법

- **유사도 유형**: 항목을 어떻게 비교하는가
  - `연속` = 수치 차이로 유사도 산출 (예: 침투율 70% vs 65%)
  - `범주` = 같은 분류면 일치, 다르면 불일치
  - `이진` = 충족/불충족 (규제 킬스위치 후보)
  - `참고` = 점수에 직접 반영 안 함, 보고서 정보용
- **출처유형**: 공식 데이터 확보 경로 (공식 출처 원칙 적용)
- **신뢰도**: 일반적으로 어느 등급으로 확보되는가 (공식/준공식/추정)
