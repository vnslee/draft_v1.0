- **유럽(NETSOL)**: 정합 **中** — NETSOL 레퍼런스 존재하나 대형 경쟁사 다수가 Alfa/Sopra/VWFS 자체구축

|S3 계약/컬렉션 방식    |순서형/범주(계약: 직접/딜러간접/온라인; 컬렉션: 단계 수)            |범주매치+순서형              |0.5           |

## 3.3 종합 점수 산식

```
진출_유사도점수(C) = Σ_k [ w_k · conf_k · sim_k(C, B) ]  /  Σ_k [ w_k · conf_k ]
```

- `w_k` : 사용자 지정 가중치(카테고리 기본값, 변경 가능) — 기획서 “사용자 지정 가중치” 요건 충족
- `conf_k` : 데이터 가용성·신뢰도(0~1, 3.2 표) — **취약 항목이 점수를 왜곡하지 않도록 자동 감쇠**
- `sim_k` : 항목별 유사도(0~1)

**카테고리 기본 가중치(합 100, 사용자 변경 가능)**

|카테고리      |기본 w|
|----------|----|
|시장(M)     |25  |
|규제(R)     |25  |
|오토파이낸스(A) |30  |
|시스템 이식성(S)|20  |

**임계값 룰(기획서 2-1/2-2 분기 자동화)**

- `점수 ≥ 기준치(예 0.70)` → **이식+확장 가능** 판정 → 베이스라인 비용 × 유사도 비율로 진출비용 추정
- `점수 < 기준치` → 경쟁사·로컬 회사 조사 모드(A3/A4 심화)
- **신뢰구간 동시 산출**: `유효신뢰도 = Σ(w_k·conf_k)/Σw_k`. 0.6 미만이면 점수에 “저신뢰” 플래그 → 단독 의사결정 금지.

## 3.4 침투율 정형화(정의 통일) — 특별 처리

4가지 정의를 **공통 분모 = 신차 판매대수**, **공통 분자 = 금융+리스 계약 대수**로 환산.

|원천 정의          |국가            |환산 방법                                                         |
|---------------|--------------|--------------------------------------------------------------|
|① 금융대수÷판매(목표형) |영국·브라질·호주(비은행)|직접 사용. 호주는 은행분 보정계수 ×(1/비은행점유율)                               |
|② 구매자/차량 비율(설문)|독일·미국         |설문 비율 ≈ 대수 비율로 근사 사용(보정 미미)                                   |
|③ 여신금액/잔액·비중   |한국·사우디·이집트·인도 |**대수 환산**: 금융취급액 ÷ 평균 차량금융액(현지 평균차값×평균 LTV) = 추정 금융대수 → ÷ 신차판매|
|④ 리스 가치(zł)    |폴란드           |리스금액 ÷ 평균 리스단가 = 리스대수 → 금융분 합산                                |

- 환산 후 각 셀에 **정의 라벨**(직접/근사/금액환산/가치환산)과 **conf**(직접 0.8 / 근사 0.7 / 금액환산 0.45 / 가치환산 0.4) 부여.
- **비교 가능성 판정**: 환산 오차가 ±10%p 초과로 추정되면 해당 국가는 침투율을 점수에서 제외하고 정성 분석으로만 사용(푸에르토리코는 미국 대리지표).

# Part 5. 주의사항(Caveats)

- **‘Oracle ADF’ 정합성 미확인**: 한국·인도·사우디 경쟁사 전부 Oracle DB+Java이나 ADF 프레임워크 명시 1차 출처 없음(Cassiopae·NFS·FinnOne 모두 Java/J2EE). 자사 타깃이 진정 ADF면 현지 인재·SI 격차 점검 필요.
- **침투율 정의 불일치**가 가장 큰 비교 리스크 — 환산·라벨링 없이는 점수 왜곡.
- **Experian은 상업기관(T4)** — 미국 사실상 표준이나 ‘공식 1차’ 아님.
- **회계연도·집계기준 차이**: 인도 SIAM(4–3월, 일부 OEM 제외), 토요타(3월 결산) 등.
- **벤더 case study 마케팅 편향**: “100% 성공”·“gold standard” 등은 사실로 단정 불가.
- **정보 공백**: 이집트 경쟁사 코어, 캐나다·푸에르토리코·브라질 로컬 코어, 사우디 ALJ/Al Rajhi/SNB, 우리·신한캐피탈 코어 미확보.
- 12개국 $110M NETSOL 계약 고객은 “leading German auto captive”로 무기명(Mercedes/BMW 추정, 미확정).
