# 에이전트 대시보드 기획

## 개요

신규 국가 진출 의사결정을 지원하는 에이전트 대시보드.

**전제 데이터**
- 이미 진출한 국가의 진출 준비 기간 / 비용 정보 보유
- 이미 진출한 국가의 시스템 정보(인프라, 상품, 코어 시스템 등) 보유

> 위 전제 데이터를 기반으로, 아래 기능들이 에이전트의 실제 타겟이다.

---

## 기능 1) 단일 국가 진출성 분석

1. 특정 **국가**를 입력받는다.
2. **비교 대상 국가**를 선정하여, 그 국가 정보(시장 · 규제 · 환경 · 시스템)의 **유사성**을 검토한다.
   비교 대상은 설정에 따라 다음 중 하나로 정해진다.
   - **(a) 동일 권역 비교** — 입력 국가와 같은 권역 내 진출 국가들을 비교 대상으로 삼는다.
   - **(b) 베이스 국가 비교** — 진출 국가 중 베이스로 지정된 일부 국가를 비교 대상으로 삼는다.
   - **2-1. 유사도 점수 ≥ 기준치** → 비교 국가의 시스템을 **이식 + 확장 가능**하다고 판단.
     - 유사도 범위에 따라 기존 진출 국가의 진출 비용을 **비율로 환산**하여 비용 추정.
   - **2-2. 유사도 점수 < 기준치** → 해당 국가의 **경쟁사 및 로컬 회사 정보**를 조사.
     - 조사 항목: 업력, 딜러망, 직원 수, 사용 솔루션 등

## 기능 2) 권역 단위 진출 순위 산정

1. 특정 **권역**을 입력받는다. (북미 / 남미 / 아태 / 유럽 / 중동)
2. 권역 내 국가들에 대해 **기능 1**을 수행한다.
3. 각 국가별 **진출 점수**를 매겨 **순위**를 산정한다.

## 서브 기능

- **근거 기반 보고서 작성** — 기능 1 / 2의 결과는 도출 근거가 포함된 보고서로 작성한다.
  - 구성: **요약 정보**(AI 인사이트 포함) + **상세 정보**(항목별 자료)
  - 출처 제약: 뉴스 · 블로그 ✕ → **국가/기관 공식 자료**(재무제표, 법령 등) ○
- **사용자 지정 가중치** — 카테고리별 우세가 달라질 수 있으므로 항목 가중치는 기본값을 두되 사용자가 변경 가능.
  - 변경된 가중치에 따라 점수 계산 결과가 달라진다.
- **조사 일자 기록** — 조사 시점마다 데이터가 변경될 수 있으므로 조사 정보에는 **조사 일자**를 포함한다.
- **가중치 룰셋 저장** — 가중치에 따라 결과가 달라지므로 보고서 생성 시 **적용된 가중치를 함께 저장**한다.

## 데이터

데이터는 총 3가지 유형으로 분류한다.

1. 국가별 기본 데이터
2. 국가별 유사도 비교를 위한 데이터
3. 비용 계산을 위한 베이스라인 국가의 비용 정보

> **(+)** 표시는 **유사성 검토 대상** 데이터. 그 외 데이터는 유사성 검토에서 제외되나, 보고서 생성 시 **상세 정보**로 작성한다.

### 1) 국가별 기본 데이터

유사도 비교에 사용되지 않는 국가의 기본 정보이자 참조 정보.

- **소속 권역** (북미 / 남미 / 유럽 / 중동 / 아태)
- **진출 정보** (진출 / 진출예정 / 미진출)
- **GDP** — 기본 출처: **IMF World Economic Outlook** (명목 GDP · 실질 성장률)

### 2) 국가별 유사성 비교를 위한 데이터

#### 2-1. 시장 정보
- GDP (USD)
- GDP 성장률 (%)
- 연간 자동차 판매 대수 **(+)**
- 자동차 판매 대수 성장률 (CAGR) **(+)**
- 오토 파이낸스/리스 침투율 (전체 차량 판매 금액의 %) **(+)**
- 오토 파이낸스 성장률 (CAGR) **(+)**
- 오토 금융 경쟁사 (해당 국가 주요 오토 파이낸스 TOP 3)
  - 토요타 파이낸스 / 폭스바겐 파이낸스는 TOP 3가 아니어도 조사
- OEM 판매량 (현대 / 기아 각각) **(+)**

#### 2-2. 규제
- **라이선스 (+)**
  - 오토파이낸스 라이선스 유무
  - 오토파이낸스 라이선스 주체
- **금융 정책 규제**
  - 금융 상품별 최소/최대 대출 기간
  - 금융 상품별 세제 혜택 여부
  - 차량 보유에 따른 세금 종류 및 계산 방식
- **정보 보호 법률 (+)**
  - 개인정보 정의 및 보유 가능 기간
  - 해외 이전 가능 여부
- **차량 관련 세제 혜택**
  - 차량 구매 시 납부 세금 종류 **(+)**
  - 차량 구매 시 납부 세금 계산식
  - 차량 구매 시 납부 세금 납부 시점
- **친환경차 시장 (+)**
  - 친환경차 수요
  - 친환경차 인프라
- **기타 (+)**
  - 외국인 대출 가능 여부
  - 채권 관련 규제

#### 2-3. 오토 파이낸스
- 오토 파이낸스 금융 침투율 **(+)**
- 신차 및 중고차 (은행 / 여신사) 이용 현황 **(+)**
- **주요 경쟁사 (은행 / 여신사)**
  - 해당 국가 TOP 3 (+ 토요타 파이낸스, 폭스바겐 파이낸스, 해당 국가 로컬 회사) **(+)**
- **주요 경쟁사 취급 상품 (+)**
  - 주력 상품: 상품 종류, 최소-최대 이율
- 주요 경쟁사 딜러십
- 주요 경쟁사 계약 방식

#### 2-4. 코어 시스템 이식성 **(+)**
- **시스템**
  - 2-3에서 조사된 경쟁사 사용 시스템
  - 해당 국가의 은행 / 여신사 솔루션
- **취급 상품**
  - 계약: 계약 방식, 연체료 처리 방식
  - 컬렉션: 단기 연체 / 중·장기 연체 처리 방식

### 3) 비용 계산용 베이스라인 데이터

베이스 국가의 진출 비용 정보. 기능 1의 **2-1**에서 유사도 비율에 따라 신규 국가 진출 비용을 환산하는 데 사용한다.
