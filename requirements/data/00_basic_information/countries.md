## 3. 제약 사항

1. `global_constraints.md` 에 정의된 제약 사항을 준수한다.
2. 인구수는 **UN World Population Prospects**(최신 revision) 중위추계를 기본 출처로 하며, 출처를 명기한다. 필요 시 각국 통계청으로 교차검증한다.
3. GDP는 **IMF World Economic Outlook**을 기본 출처로 하며(명목 GDP·실질 성장률), 기준연도와 출처를 명기한다. 필요 시 각국 통계청으로 교차검증한다.

---

## 4. 산출물 생성 프로세스

- 조사한 국가 정보를 **JSON 초안**으로 생성한다.
- **산출물 경로**: `mockup/countries/` 폴더에 생성한다. (폴더가 없으면 생성)
- JSON 구조는 **초안 수준**으로 두고, 추후 DB 스키마가 확정되면 정규화 / 테이블 구조 등 데이터 형태를 조정한다.
- **미확보 데이터는 채워넣지 않는다** (누락 = 미확보). 점유율(share)은 출처에 명시된 경우만 기재한다.
- 산출물에는 **데이터 기준연도**와 **출처**(인구 / 자동차 / GDP)를 메타 정보로 함께 기록한다.
- 신뢰성 정보가 필요한 데이터 필드는 **`{ value, year, estimated, source, note? }`** 메타 구조로 저장한다.
    - `estimated`: 추정치/미확정이면 `true`, 확정 실측치면 `false`
    - agent가 표시 시점에 **연도 / 추정 여부 / 출처**를 함께 보여줄 수 있도록 한다.
    - 베스트셀러처럼 Top N 목록은 **`complete`** 플래그로 목록 완성 여부를 표기한다.
